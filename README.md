# MoneyCollect Java client library

## Installation

### Requirements

- Java 1.8 or later

### Maven users

Add this dependency to your project's POM:

```xml
<dependency>
  <groupId>com.moneycollect</groupId>
  <artifactId>moneycollect-java</artifactId>
  <version>1.0.0</version>
</dependency>
```
